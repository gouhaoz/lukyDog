# LukyDog(1.0Pro)抽奖程序

这是一个基于 Electron + React（antd） 框架的 xlsx 表格抽奖程序；只适用于在 Windows 系统上进行抽奖活动。

## 安装包下载

- [Windows 最新版本（点击下载）](https://github.com/gouhaoz/lukyDog/releases/download/1.0.0/LukyDog.1.0Pro.Setup.6.0.0.exe.zip)

### 开发环境启动命令：

npm：

```bash
npm run electron:dev
```

yarn：

```bash
yarn electron:dev
```

### 生产环境打包命令：

npm：

```bash
npm run electron:build
```

yarn：

```bash
yarn electron:build
```

## 主要技术栈

- Electron
- React
- Ant Design Pro
- TypeScript
- Node.js
- SqLite3
